## 1. Github 계정 만들기
1) [Git](https://github.com/)에서 이메일 주소를 이용하여 무료계정 만들기
2) 계정을 만들고, 오른쪽 위의 내 프로파일 이미지를 눌러서 `Your repositories`로 가보기
